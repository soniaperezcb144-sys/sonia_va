Abrir Imagen con Python

Código del Proyecto

import cv2 # Cargar una imagen img = cv2.imread('gato.jpg') # Mostrar la imagen en una ventana cv2.imshow('Mi Imagen', img) # Esperar hasta que el usuario presione una tecla cv2.waitKey(0) cv2.destroyAllWindows()

Glosario de Funciones

abrir_imagen_pil(ruta_imagen)
Función que utiliza la biblioteca PIL (Pillow) para abrir y mostrar una imagen. Es ideal para procesamiento básico de imágenes y soporta múltiples formatos.
  • ruta_imagen: String con la ruta del archivo de imagen a abrir
  • return: Objeto Image de PIL con la imagen cargada
abrir_imagen_cv2(ruta_imagen)
Función que emplea OpenCV (cv2) para cargar y visualizar imágenes. OpenCV es más eficiente para operaciones de visión artificial y procesamiento en tiempo real.
  • ruta_imagen: String con la ruta del archivo de imagen
  • return: Array de NumPy representando la imagen en formato BGR
visualizar_imagen(ruta_imagen)
Función que usa Matplotlib para mostrar imágenes en un entorno más científico. Permite crear visualizaciones más complejas y gráficos con las imágenes.
  • ruta_imagen: String con la ruta del archivo a visualizar
  • return: None (muestra la imagen directamente)
Image.open()
Método de PIL que abre un archivo de imagen y crea un objeto Image. Soporta formatos como JPG, PNG, BMP, GIF, entre otros.
cv2.imread()
Método de OpenCV que lee una imagen desde un archivo y la retorna como un array de NumPy. Lee las imágenes en formato BGR por defecto.
cv2.imshow()
Muestra una imagen en una ventana de OpenCV. Requiere usar waitKey() para mantener la ventana abierta.
