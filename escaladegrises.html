Código del Proyecto

import cv2 img = cv2.imread('equipo.jpg') #devuelve una matriz NumPy que representa la imagen en formato BGR (Azul, Verde, Rojo). gris = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)#Convierte la imagen de color (BGR) a escala de grises. cv2.imshow('Original', img) #Muestra una ventana con el nombre "Original" que contiene la imagen a color. cv2.imshow('Escala de Grises', gris)#Muestra otra ventana con el título "Escala de Grises", cv2.waitKey(0)#El parámetro 0 significa “esperar indefinidamente”. #Si pusieras un número, por ejemplo 1000, esperaría 1000 milisegundos (1 segundo) antes de cerrar las ventanas automáticamente.""" cv2.destroyAllWindows()

Glosario de Funciones

cv2.imread(filename)
Lee una imagen desde un archivo y la devuelve como una matriz NumPy. La imagen se carga en formato BGR (Azul, Verde, Rojo), que es el formato por defecto de OpenCV.
  • filename: String con el nombre o ruta del archivo de imagen
  • return: Matriz NumPy tridimensional (altura, ancho, canales BGR)
cv2.cvtColor(img, code)
Convierte una imagen de un espacio de color a otro. En este caso, usa cv2.COLOR_BGR2GRAY para convertir de BGR (color) a escala de grises usando la fórmula estándar: Y = 0.299*R + 0.587*G + 0.114*B
  • img: Imagen de entrada (matriz NumPy)
  • code: Código de conversión (ej: COLOR_BGR2GRAY)
  • return: Imagen convertida en el nuevo espacio de color
cv2.imshow(winname, mat)
Muestra una imagen en una ventana específica. Crea una ventana con el nombre especificado y visualiza la imagen proporcionada. La ventana se ajusta automáticamente al tamaño de la imagen.
  • winname: String con el nombre de la ventana
  • mat: Imagen a mostrar (matriz NumPy)
  • return: None
cv2.waitKey(delay)
Espera a que el usuario presione una tecla durante un tiempo específico. Es necesaria para mantener las ventanas abiertas y capturar eventos del teclado.
  • delay: Tiempo de espera en milisegundos. Si es 0, espera indefinidamente
  • return: Código ASCII de la tecla presionada, o -1 si no se presionó ninguna
cv2.destroyAllWindows()
Cierra todas las ventanas de OpenCV que se hayan creado durante la ejecución del programa. Libera los recursos asociados a las ventanas. Es una buena práctica llamarla al finalizar.
  • return: None
cv2.COLOR_BGR2GRAY
Constante que indica el tipo de conversión de color a realizar. Convierte una imagen de 3 canales BGR (Azul, Verde, Rojo) a 1 canal en escala de grises. La conversión utiliza ponderaciones específicas para cada canal que se ajustan a la percepción humana.
Formato BGR
OpenCV utiliza el orden BGR (Azul, Verde, Rojo) en lugar de RGB (Rojo, Verde, Azul). Esto es importante al trabajar con otras bibliotecas como PIL o Matplotlib que usan RGB. Cada canal tiene valores de 0 a 255.
Escala de Grises
Representación de una imagen usando un solo canal de intensidad. Cada píxel tiene un valor entre 0 (negro) y 255 (blanco). Reduce el tamaño de la imagen en memoria y es útil para muchos algoritmos de procesamiento de imágenes.
